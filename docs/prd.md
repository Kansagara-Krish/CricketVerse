# CricketVerse AI - Functional Specification

## Project Title

**CricketVerse AI: An Intelligent Mobile Application for Live Cricket Scoring, AI Commentary, Match Prediction, and Real-Time Analytics**

---

# User Roles

## 1. Admin
System owner who manages tournaments, teams, players, and application settings.

## 2. Match Scorer
Official responsible for updating live match events and scores.

## 3. Cricket Fan (User)
End user who follows matches, commentary, predictions, and statistics.

---

# Admin Pages

| Page | Purpose | Main Features |
|------|---------|---------------|
| Dashboard | Overview | Live stats, quick actions, recent matches, notifications |
| Tournament Management | Create/Edit tournaments | Add, edit, delete tournaments |
| Team Management | Manage teams | Add/edit teams, logos, search |
| Player Management | Manage players | Add/edit players, player profiles |
| Schedule Match | Create fixtures | Select teams, venue, date, match type |
| Live Scoring | Update live match | Runs, wickets, extras, over control, undo |
| AI Commentary | Preview commentary | Auto-generate text, voice toggle |
| Prediction | Analytics | Win probability, projected score |
| Notifications | Broadcast | Send alerts to users |
| Profile & Settings | Account | Edit profile, theme, logout |

---

# Match Scorer Pages

| Page | Purpose | Main Features |
|------|---------|---------------|
| Assigned Matches | View fixtures | Open assigned match |
| Live Score | Ball-by-ball update | Runs, wickets, extras, batsmen, bowler |
| Commentary | Generated commentary | Auto commentary preview |
| Match Summary | Finish match | Result and summary |
